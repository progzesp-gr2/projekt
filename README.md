## Informacje o Rolach w projekcie

Product Owner:

- tworzy projekty
- dodaje ludzi
- zarządza backlogiem
- może wybrać Scrum Mastera

Scrum Master:

- zarządza sprintami
- tworzy / przypisuje zadania
- pilnuje pracy zespołu

Programmer:

- widzi swój projekt
- widzi swoje zadania
- widzi członków zespołu
